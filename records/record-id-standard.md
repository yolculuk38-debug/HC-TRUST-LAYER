# Record ID Standard

This document defines the standard format used for Humanity Chain interaction record identifiers.

## Purpose

Record IDs are used to uniquely identify archived interaction records across the Humanity Chain system.

## Recommended Format

Current preferred structure:

HC-[MODEL]-[YEAR]-[NUMBER]

Examples:

- HC-CHATGPT-2026-0001
- HC-GEMINI-2026-0002
- HC-CLAUDE-2026-0003

## Components

- HC → Humanity Chain prefix
- MODEL → AI model or source
- YEAR → Record year
- NUMBER → Sequential archive number

## Principles

Record IDs should:

- remain unique
- remain permanent
- remain traceable
- remain human-readable

## Usage

Record IDs may appear in:

- archive records
- hash references
- QR references
- timeline entries
- verification pages
- public archive indexes
