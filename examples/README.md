# Witness Examples

This directory contains example witness records for HC-TRUST-LAYER.

These examples demonstrate how AI responses, confirmations, and interaction records may be documented inside the protocol.

The files in this directory are demonstration records only.

---

## Example Types

- AI response examples
- Multi-model confirmations
- Human witness records
- Verification examples
- Timeline-linked records

---

Purpose:
To help contributors understand the witness structure and record format.
