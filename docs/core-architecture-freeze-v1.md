# HC:// Core Architecture Freeze v1

## Status

This document marks the first HC:// core architecture freeze baseline.

It summarizes the stable protocol direction after the trust infrastructure expansion from PR 100 through PR 149.

---

# Identity

HC:// is an open verification protocol and digital trust infrastructure layer.

It is not:
- a social network
- a token system
- a traditional blockchain
- a simple AI detector

It is designed as:
- provenance infrastructure
- explainable verification system
- portable trust validation layer
- human + AI hybrid trust architecture
- adversarial-aware verification protocol
