# Contributing

Thank you for contributing to HC-TRUST-LAYER (provenance origin archive ecosystem).

This project is an experimental open archive and verification protocol focused on transparent human-AI interaction records.

## Contribution Rules

- Keep records factual and traceable.
- Avoid unverifiable claims.
- Do not impersonate AI systems or people.
- Respect archive structure and formatting.
- Major changes should be discussed before merge.

## Suggested Workflow

1. Open an Issue.
2. Describe the proposal or record.
3. Submit changes through commit or pull request.
4. Archive verified material only.

## Witness Structure

Witnesses may include:
- Human reviewers
- AI-generated interaction references
- Linked archive records
- Hash verification references

## Archive Principles

- Transparency before narrative.
- Records before claims.
- Verification before distribution.

## Status

Open contribution model under development.
