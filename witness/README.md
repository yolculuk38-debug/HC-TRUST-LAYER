# Witness Layer

This directory contains witness-related records and references for HC-TRUST-LAYER.

## Purpose

The witness layer is designed to document responses, observations, and confirmations from different AI models or independent contributors.

## Witness Types

A witness entry may come from:

- AI models
- human reviewers
- independent contributors
- archive maintainers

## Witness Entry Format

Each witness entry may include:

- Witness ID
- Record ID
- Model or contributor name
- Date
- Prompt
- Response or observation
- Verification status
- Related hash reference
- Related timeline entry

## Principle

Witness entries are not treated as absolute proof.

They are structured observations that support transparency, comparison, and historical documentation.

## Usage

Witness references may be connected to:

- records
- hash references
- QR references
- timeline entries
- verification pages
